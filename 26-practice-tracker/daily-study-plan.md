# Daily Study Plan

## Daily Routine

1. Read one topic note.
2. Run one code example.
3. Write one summary in your own words.
4. Complete two exercises.
5. Answer five interview questions.
6. Commit progress with Git.

## Daily Reflection

- What did I learn?
- What broke?
- How did I debug it?
- Can I explain it in two minutes?
- What should I revise tomorrow?
