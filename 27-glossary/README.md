# 27 Glossary

This module is the beginner glossary for the AI Engineer JRSS Reskilling Course.

Use `glossary.md` when you meet a term and need three things quickly:

- simple meaning
- small example
- where it appears in AI Engineer work

The glossary is not only definitions. Each term is written so a beginner can connect the word to real modules such as FastAPI, RAG, MCP, LangGraph, Docker, Kubernetes, and the final POC.
