# 14 Agents and Multi-Agent Systems

## What this module is

This module teaches what AI agents are and when they are useful.

Simple meaning:

```text
Agent = AI workflow that can use a model, state, tools, and decisions to complete a task
```

## Why it matters

Many beginner demos call everything an agent. In real AI Engineering, you should know when an agent is useful and when a simple prompt or function is enough.

Agents matter when a workflow needs:

- tool calling
- planning
- execution
- review
- state
- routing
- handoff between specialists

## What you should finish knowing

You should understand LLM vs agent, tool-using agent, planner, executor, reviewer, supervisor, A2A agent-to-agent awareness, handoff, memory basics, multi-agent systems, and when not to use agents.
