# 19 Langflow Basics

## What this module is

Langflow is a visual flow builder for AI workflows.

Simple meaning:

```text
Langflow = drag-and-connect interface for building AI flows
```

## Why it matters

Visual builders help beginners and stakeholders understand AI pipelines such as prompt -> LLM -> output or retriever -> prompt -> LLM.

## What you should finish knowing

You should understand nodes, edges, prompt node, LLM node, retriever node, RAG flow, where Langflow fits, and its limitations.

## Study order

1. `00-overview.md`
2. `01-nodes-rag-flow.md`
3. `exercises.md`
4. `cheatsheet.md`
5. `interview-questions.md`

## Practical scope

This module is awareness-level. Installation can be heavy, so the focus is conceptual flow design.

## What not to over-focus on

Do not treat Langflow as a replacement for understanding Python, APIs, RAG, and prompts.

## How this helps

It helps you explain visual prototyping and compare visual flows with code-based systems.
