# Langflow Interview Questions

## What is Langflow?

Short answer:

Langflow is a visual builder for AI workflows.

Expanded explanation:

It lets users connect nodes such as prompt, LLM, retriever, and output components to prototype flows.

Real project example:

A RAG prototype can be shown as document loader -> vector store -> retriever -> prompt -> LLM.

Common wrong answer:

"Langflow is the model."

When to say this in interview:

Use when asked about visual AI builders.

## Where does Langflow fit?

Short answer:

It is useful for prototyping and explaining flows.

Expanded explanation:

It helps visualize components, but production systems may still need Python code, FastAPI, tests, logs, and deployment.

Real project example:

Prototype RAG in Langflow, then implement backend in Python for the final POC.

Common wrong answer:

"Langflow replaces all backend code."

When to say this in interview:

Use when asked prototype vs production.
