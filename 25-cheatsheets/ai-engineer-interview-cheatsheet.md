# AI Engineer Interview Cheatsheet

## Role Explanation

Meaning: AI Engineer integrates AI into applications.

Use when: Asked about target role.

Example: "I build APIs, prompts, RAG workflows, tool integrations, validation, and deployment around models."

Be careful: Do not say the role is only model training.

## POC Explanation

Meaning: Explain use case plus architecture.

Use when: Asked "Tell me about your project."

Example: "AI QA assistant using FastAPI, Pydantic, RAG, mock LLM, MCP-style tools, and LangGraph-style workflow."

Be careful: Avoid saying only "chatbot."

## Production Readiness

Meaning: Security, reliability, deployment, monitoring.

Use when: Asked "How would you productionize it?"

Example: auth, secrets, logs, retries, evaluation, Docker, Kubernetes.

Be careful: POC is not production by default.
