# IBM AI Engineer Awareness Interview Questions

## 1. What does an AI Engineer do?

Builds, integrates, configures, and validates AI systems inside applications.

## 2. What enterprise concerns matter for GenAI?

Privacy, security, auditability, explainability, logging, cost, and governance.

## 3. How do you secure API keys?

Use environment variables or secret managers. Never hardcode or commit real keys.

## 4. How do you explain a POC?

Start with problem, architecture, demo flow, technical components, limitations, and next steps.

## 5. What is productionization?

Turning a working demo into a secure, reliable, monitored, scalable application.
