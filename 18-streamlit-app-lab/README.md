# 18 Streamlit App Lab

## 1. What This Module Is

This module teaches Streamlit basics for building simple AI demo UIs.

## 2. Why It Matters

POC demos need a simple interface. Streamlit lets you build one quickly with Python.

## 3. What The Learner Should Finish Knowing

You should understand Streamlit app files, text input, buttons, displaying output, calling a FastAPI backend, and building a mock AI test case generator UI.

## 4. Study Order

- `interview-questions.md`: demo/interview Q&A

## 6. Practical Scope

The UI uses mock output first. Backend connection is shown as an optional pattern.

## 7. What Not To Over-Focus On

Do not over-focus on advanced frontend styling. Streamlit is for quick demos, not complex production UI.

## 8. How This Helps

Streamlit helps you present the final POC in a stand-and-deliver demo.
