# Streamlit Interview Questions

## What is Streamlit?

Short answer:

Streamlit is a Python tool for building quick data and AI demo UIs.

Expanded explanation:

It lets you create text inputs, buttons, and output sections with simple Python code.

Real project example:

The POC can use Streamlit as a UI for entering requirements and viewing generated test cases.

Common wrong answer:

"Streamlit replaces FastAPI."

When to say this in interview:

Use when explaining demo frontend choices.

## Streamlit vs FastAPI?

Short answer:

Streamlit is UI-focused. FastAPI is backend/API-focused.

Expanded explanation:

Streamlit is good for demos. FastAPI is better for API endpoints, validation, integration, and deployment as a backend service.

Real project example:

Streamlit sends requirement text to FastAPI `/generate-test-cases`.

Common wrong answer:

"They do exactly the same thing."

When to say this in interview:

Use when asked architecture choices.
